Small, sharp tools

Single-file utilities for product work under uncertainty — estimation, prioritisation, and the conversations around them. No accounts, no tracking, no build step; each tool is one HTML file and your data stays in the URL.

Fermi estimator /fermi

Back-of-the-envelope arithmetic that's honest about what you don't know. Every input is a 90% range; the answer is a P10/P50/P90 distribution — with threshold probability, A/B scenarios, and a read on which input is worth researching first.

Rank stability /rank

Weighted prioritisation scores are guesses wearing a suit. This wobbles your weights and scores over thousands of simulations and reports which ranks survive — and which are ties you shouldn't argue about.

Roadmap as code /roadmap

Write your roadmap as plain text; get a deck-ready now/next/later (or monthly/quarterly) graphic with snapshot-to-snapshot change tracking, SVG/PNG export, and zero box-nudging.
